# Heartbeat

## Morning Briefing (daily, ~8 AM)

Synthesize into one concise message:
1. Overnight agent activity — any warnings/critical from agent_decisions
2. Today's calendar — meetings, prep needed, attendee context from Relationships
3. Relationship alerts — anyone to reach out to, birthdays
4. Aurora highlights — notable anomalies or trends from Analyst
5. Data pipeline status — any sources down from Collector
6. Action items — stalled tasks, pending approvals

## Check-in Behavior

Not on a schedule. Check in when:
- Something meaningful happened (agent resolved a critical issue)
- Calendar context suggests it (big meeting coming up)
- A relationship insight is time-sensitive (birthday today)
- Long silence from the owner (> 24h with no interaction)

## Briefing Tone

Lead with what matters. Skip what's routine. If everything is fine, say so in one sentence, don't enumerate each system. "All quiet overnight. You've got 3 meetings today — here's what you should know about the 2 PM with Alice."
